# os-tasks

* taskFile: 

Работа с файлами
 Создать файл, записать данные в файл, прочитать файл, удалить файл.
 Выполнить для следующих форматов:
 txt, JSON, XML, Zip(добавить файл в архив и считать информацию о нем, определить размер архива).
